# 群像-单人图一致性检查清单

在完成系列生成后使用此清单进行质量检查。

## 检查流程

```
群像生成完成 → 创建位置映射表 → 生成单人图 → 逐一对比检查 → 修正不一致项
```

## 一致性检查项

### ✅ 发色一致性 (最关键!)

- [ ] 对比群像和每张单人图中的发色
- [ ] 确认发色名称和 Hex 色号匹配
- [ ] 如发现不一致：
  - 检查单人图 Prompt 是否遗漏发色声明
  - 在 Prompt 开头添加强调：`Hair: MUST BE [颜色] [Hex]`
  - 添加负向提示：`NOT brown hair, NOT dark hair`

### ✅ 眼睛一致性

- [ ] 眼睛大小比例是否一致
- [ ] 眼型是否一致（圆形/杏仁形等）
- [ ] 高光位置和大小是否一致
- [ ] 瞳色是否一致

### ✅ 比例一致性

- [ ] 头身比是否一致
- [ ] 脸型是否一致
- [ ] 身体胖瘦是否一致

### ✅ 服装匹配

- [ ] 服装颜色是否与设计一致
- [ ] 图案/装饰是否正确
- [ ] 配件是否齐全

### ✅ 表情匹配

- [ ] 表情是否符合款式设计
- [ ] 腮红位置和颜色是否一致

---

## 常见问题排查

| 问题 | 可能原因 | 解决方法 |
|-----|---------|---------|
| 发色从金变棕 | Prompt中未声明发色 | 显式添加发色+Hex |
| 眼睛变小 | 未锁定眼睛比例 | 添加 "eyes 42% of face width" |
| 头身比变化 | 未锁定比例 | 添加 "head-to-body ratio 2.3:1" |
| 服装颜色偏差 | 描述不够精确 | 使用 Hex 色号 |
