# 角色锚点模板

此模板用于定义角色的固定特征，确保群像和单人图保持一致。

## 使用说明

1. 复制此模板到你的系列目录
2. 填写所有 `[占位符]` 内容
3. 在生成每张图片时，完整复制 LOCKED 部分到 Prompt 中
